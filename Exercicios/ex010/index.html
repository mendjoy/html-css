Usando Links externos

Você pode acessar o meu perfil no GitHub.

Para utilizar links externos é necessário utilizar a tag <a></a> e dentro dela utilizar 'href=""', colocando dentro das aspas a url a ser direcionada. Quando utilizado links externos é recomendado que o link seja aberto em uma nova guia, para isso, dentro da tag após a url é colocado 'target="_blank"', também pode ser utilizada dentro da tag 'rel="external" para indicar que é um link externo.

Usando Links Internos

Esta é a primeira página do site, se você quiser pode acessar também a minha Segunda página

Links Internos, são links que serão direcionados a outras áreas de um mesmo Site, utiliza-se a mesma tag utilizada para os links externos, porém é utilizado dentro da tag 'rel="next"' na pagina principal e na segunda pagina, para voltar para a anterior pode ser utilizado 'rel="prev"'. O target utilizado para que o link seja aberto na mesma pagina é 'target="_self"'
