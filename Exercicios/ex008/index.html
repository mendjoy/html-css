Principais formatações

Negrito/Destaque

Existem duas maneiras de colocar o texto em negrito, uma semântica e outra maneira não semântica (<b>, não é uma tag semântica).

Nesta frase temos um termo em negrito usando a tag <b>.

Nesta frase, tenho um termo em destaque Usando a tag STRONG (<strong> é uma tag semântica).


Itálico/Ênfase

Nesta frase temos um termo em itálico, usando a tag <i> (não semântica).

Nesta frase, temos um Termo em ênfase usando a tag <em> (semântica).


Texto marcado

Podemos também criar texto marcado usando a tag <mark>.


Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo. Usando a tag Obsoleta <big> para textos grandes e a tag <small> para textos pequenos (não está obsoleta).


Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado utilizando a tag <del>.


Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.

Existe também o sublinhado utilizando a tag <u> que é não semântica.


Texto sobrescrito

Para inserir coisas do tipo: x20+3, utilizando a tag <sup>.


Texto subscrito

Para inserir coisas do tipo: H2O, utilizando a tag <sub>.
