Para criar listas ordenadas (Possuem ordem/numeração) é utilizada a tag <ol></ol>, cada item dessa lista pode ser colocado dentro da tag <li> (</li> opcional). Dentro da tag <ol></ol> pode ser utilizada o parâmetro "type" que possui 5 opções Números, letras maiúsculas, miniscúlas, algarismos romanos minúsculos e maiscúlos. Para escolher o número inicial das númerações é utilizado o parâmetro "start".
