Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Para manter as letras de um código espaçadas para uma melhor visualização podemos utilizar a tag <code>.

Exemplo:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O numero {num} é PAR') else: print(f'O número {num} é IMPAR') print('Fim do programa')

Podemos observar que a tag <code> não mostra o código com indentação. Para organizar os códigos utilize a tag <pre> envelopando a tag <code>.

Exemplo:

Segundo Jeff Noble, no seu livro HTML para leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto

Para Fazer Citações completas, utilizar a tag <blockquote>, esta faz com que a citação seja deslocada para a direita, dentro desta tag é possivel colocar o "cite="" e dentro das aspas o link do local onde se encontra a citação,isso faz com que os mecanismos do google saibam onde está disponivel o texto e manter os direitos dos autores.
